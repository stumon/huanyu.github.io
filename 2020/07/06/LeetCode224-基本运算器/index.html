LeetCode 224 基本运算器

题目

实现一个基本的计算器来计算一个简单的字符串表达式的值。

字符串表达式可以包含左括号 ( ，右括号 )，加号 + ，减号 -，非负整数和空格。

示例

>输入: "(1+(4+5+2)-3)+(6+8)"
>输出: 23

思路

把中缀表达式转成后缀表达式，这道题只要求加减，没有乘除，所以只要栈不为空，那么运算符进来时，栈里的运算符就要出栈。还要注意括号，如果是左括号，就直接进栈，如果是右括号，栈里元素要出栈到栈顶元素为左括号为止。

中缀表达式转后缀表达式的核心思路就是栈+链表。栈用来存运算符和括号，链表用来存后缀表达式

后缀表达式用栈计算。碰到运算符，比较优先级（栈顶元素优先级小于当前运算符，当前运算符就进栈，否则栈顶元素就出栈）

代码

class Solution {
    public int calculate(String s) {
        return calBackExp(transfetToBackExp(s));
    }
    //根据后缀表达式计算最终结果
    private int calBackExp(LinkedList<String> list)
    {
        Stack<Integer> stack=new Stack<>();
        for(String s:list)
        {
            if(isNumerical(s)){
                stack.push(Integer.parseInt(s));
            }else if(s.equals("+") || s.equals("-")){
                int n1=stack.pop();
                int n2=stack.pop();
                if(s.equals("+")) stack.push(n1+n2);
                else stack.push(n2-n1);
            }
        }
        return stack.pop();
    }

	//把前缀表达式转成后缀表达式
    private LinkedList<String> transfetToBackExp(String exp){
        LinkedList<String> list=new LinkedList<>();
        Stack<String> stack=new Stack<>();
        int len=exp.length();
        for(int i=0;i<len;i++)
        {
            char c=exp.charAt(i);
            if(c==' ') continue;//空格就继续循环
            else if(isDigit(c))//数字就进入链表
            {
                int k=i+1;
                for(;k<len && isDigit(exp.charAt(k));k++){}
                list.add(exp.substring(i,k));
                i=k-1;
            }else if(c=='('){
                stack.push("(");
            }else if(c==')'){
                while(!stack.empty() && stack.peek()!="(")
                {
                    list.add(stack.pop());
                }
                stack.pop();
            }else if(c=='+' || c=='-'){
                while(!stack.empty() && !stack.peek().equals("(")){
                    list.add(stack.pop());
                }
                
                stack.push(c+"");
            }
        }
        
        while(stack.size()>0) list.add(stack.pop());
        return list;

    }
    private boolean isDigit(char c)
    {
        if(c-'0'>=0 && c-'0'<=9) return true;
        return false;
    }
    private boolean isNumerical(String str)
    {
        for(int i=0;i<str.length();i++)
        {
            if(isDigit(str.charAt(i))==false)  return false;
        }
        return true;
    }
}
