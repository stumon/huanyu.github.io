LeetCode 43 字符串相乘

题目

给定一组非负整数，重新排列它们的顺序使之组成一个最大的整数。

示例 1:

>输入: [3,30,34,5,9]
>输出: 9534330

思路

网上最多的做法，就是使用语言自带的排序功能，然后自己写一个排序规则，就比如下面这种

class Solution {
    public String largestNumber(int[] nums) {
        int len=nums.length;
        if(len==0) return "";
        String[] tmp=new String[len];
        for(int i=0;i<len;i++) tmp[i]=nums[i]+"";
        Arrays.sort(tmp,new Comparator<String>(){
            @Override
            public int compare(String s1,String s2)
            {
                String t1=s1+s2;
                String t2=s2+s1;
                return -t1.compareTo(t2);
            }
        });
        if(tmp[0].charAt(0)=='0') return "0";
        StringBuilder sb=new StringBuilder();
        for(int i=0;i<len;i++) sb.append(tmp[i]);
        return sb.toString();
    }
}

2020年7月面试shoppe时，面试官拒绝我使用这种写法，然后我面试就翻车了。。。

回来后，查了一下，发现可以用冒泡排序，两两比较，这样每一次都可以选出来最大的一个元素放在队首。时间复杂度是O(long(n^2))。详细过程过程在下面

代码如下

class Solution {
    public String largestNumber(int[] nums) {
        StringBuilder sb=new StringBuilder();
        int len=nums.length;
        if(len==0) return "";
        String[] strs=new String[len];
        for (int i=0;i<len;i++) strs[i]=nums[i]+"";
        for (int i=0;i<len;i++)
        {
            for(int j=i+1;j<len;j++)
            {
                //解释下，输入: [3,30,34,5,9]
                //正常顺序是 330，反序是303，前者大于后者，不需要改变，继续迭代
                //正常顺序是334，反序是343，前者小于后者，交换位置[34,30,3,5,9]，继续迭代
                //正常顺序是345，反序是534，前者小于后者，交换位置[5,30,3,34,9]，继续迭代
                //正常顺序是59，反序是95，前者小于后者，交换位置[9,30,3,34,5]，继续迭代
                //这样一轮迭代完，可以确定第一个位置是最大的位置。然后依次迭代。最终把数组转成字符串即可
                String s1=strs[i]+strs[j];//正常顺序
                String s2=strs[j]+strs[i];//反序
                if (s1.compareTo(s2)<0)//如果正常顺序小于反序，就交换两者
                {
                    String tmp=strs[i];
                    strs[i]=strs[j];
                    strs[j]=tmp;
                }
            }
        }
        for (String s:strs) sb.append(s);
        String res=sb.toString();
        if(res.charAt(0)=='0') return "0";
        return res;
    }
}
