LeetCode106 从中序和后序遍历序列中构建二叉树

先从后序遍历中得到根节点，得到根节点在中序遍历中的位置index，中序遍历序列中index左边的就是左子树，右边的就是右子树，然后递归求解左右子树即可
难点：如何计算下一次后序遍历左子树根节点的位置（后序遍历根节点的下标-中序遍历根节点右边的子节点数量=下一次迭代的后序遍历根节点位置）
中序+后序构建二叉树：难点是计算左子树的位置。因为后序根节点的位置-1就是右子树的位置
中序+先序构建二叉树：难点是计算右子树的位置，因为先序根节点的位置+1就是左子树的位置。
当然这两种情况都要判断边界

java代码如下：

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public TreeNode buildTree(int[] inorder, int[] postorder) {
        HashMap<Integer,Integer> map=new HashMap<>();
        int length=postorder.length;
        for(int i=0;i<length;i++)
        {
            map.put(inorder[i],i);
        }
        return help(inorder,postorder,0,length-1,length-1,map);
    }
    TreeNode help(int[] inorder, int[] postorder, int inStart, int inEnd, int postEnd, HashMap<Integer,Integer> map)
    {
        if(postEnd<0 || inStart>inEnd) return null;
        int index=map.get(postorder[postEnd]);
        TreeNode node=new TreeNode(postorder[postEnd]);
        node.left=help(inorder,postorder,inStart,index-1,postEnd-(inEnd-index)-1,map);
        node.right=help(inorder,postorder,index+1,inEnd,postEnd-1,map);
        return node;
    }
}
