计数排序

计数排序是通过数组中元素出现的次数来计数的，先找到数组中的最大值max，然后建立一个长度为max+1的数组count，对原数组nums中的数字出现的次数遍历，最后再遍历count数组，如果count[i]值不为空，说明nums中有这个值，输出回原数组nums，遍历完count数组，即得到了nums数组排序后的结果

代码如下

void countSort(int[] nums)
{
    int res=0;//记录数组最大值
    int len=nums.length;
    for(int i=0;i<len;i++)
    {
        if(res<nums[i]) res=nums[i];//找到数组最大值
    }
    int[] count=new int[res+1];//建立长度为res+1的计数数组
    for(int i=0;i<len;i++) count[nums[i]]++;//计算nums[i]出现的次数
    int k=0;
    for(int i=0;i<res;i++)//遍历count数组
    {
        while (count[i]>0)//如果数组大于0，说明nums数组中有i这个值
        {
            nums[k++]=i;//将这个值写会到数组中
            count[i]--;//i这个值出现次数-1
        }
    }
}

void swap(int[] nums, int i, int j)
{
    if(i>=nums.length) return;
    int temp=nums[i];
    nums[i]=nums[j];
    nums[j]=temp;
}

优化：

容易发现count数组小于下标nums中最小值的空间都被浪费了，是不可能存在值的，所以可以建立一个长度为Max-min+1数组

代码如下：

void countSort(int[] nums)
{
    int max=nums[0];
    int min=nums[0];
    int len=nums.length;
    for(int i=0;i<len;i++)
    {
        if(max<nums[i]) max=nums[i];
        if(min>nums[i]) min=nums[i];
    }
    int[] count=new int[max-min+1];//优化空间
    for(int i=0;i<len;i++) count[nums[i]-min]++;//优化代码
    int k=0;
    for(int i=0;i<max-min+1;i++)
    {
        while (count[i]>0)
        {
            nums[k++]=i+min;//优化代码
            count[i]--;
        }
    }
}

void swap(int[] nums, int i, int j)
{
    if(i>=nums.length) return;
    int temp=nums[i];
    nums[i]=nums[j];
    nums[j]=temp;
}
