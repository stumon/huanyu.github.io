基数排序

基数排序是桶排序的升级版把数组中每一个数字，都分成不同位，先比较数字不同位的大小，先按照个位大小排序，再按照十位大小排序….一直到最高位，最高位的排序结果就是最终的排序结果，具体见下图，图片来自@sky。

Java代码如下：

void buicketSort(int[] nums)
{
    int maxExp=getExp(nums);
    int len=nums.length;
    ArrayList<ArrayList<Integer>> lists=new ArrayList<>();
    for (int i=0;i<10;i++) lists.add(new ArrayList<>());//因为是十进制，所以有0~9，十个桶
    for (int i=1;i<=maxExp;i++)
    {
        //1234,假如想得到百位的2，应该是1234%1000=234,234/100=2，所以应该先取模，再相除，最终就知道了值
        double exp1=Math.pow(10, i);
        double exp2=Math.pow(10,i-1);
        for (int j=0;j<len;j++)//先按照位添加到指定数组中
        {
            int index=(int)(nums[j]%exp1/exp2);//算出这数字nums[j]的某一位的值，来决定添加到哪一个桶上
            lists.get(index).add(nums[j]);//添加到指定桶
        }
        int k=0;//下标，按照位排好序的结果要写回原数组
        for(int j=0;j<10;j++)//遍历list中的每一个数组，将其写回到原数组中去
        {
            while (lists.get(j).size()>0)
            {
                ArrayList<Integer> temp=lists.get(j);
                nums[k++]=temp.get(0);//把按照某一位排序的结果写回到数组
                temp.remove(0);
            }
        }
    }
}

用处：主要用于处理数字较多的情况，比如手机号排序。
