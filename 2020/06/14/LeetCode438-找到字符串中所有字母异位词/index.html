LeetCode438-找到字符串中所有字母异位词

题目

给定一个字符串 s 和一个非空字符串 p，找到 s 中所有是 p 的字母异位词的子串，返回这些子串的起始索引。

字符串只包含小写英文字母，并且字符串 s 和 p 的长度都不超过 20100。

>输入:
>s: "abab" p: "ab"

>输出:
>[0, 1, 2]

说明：

  • 字母异位词指字母相同，但排列不同的字符串。
  • 不考虑答案输出的顺序

注意：相同字母也是异位词，如”ab”也是”ab”的异位词

思路

维护一个字符串P长度的滑动窗口，不断移动判断即可

代码

class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        int sLen=s.length(),pLen=p.length();
        int left=0,right=0;
        HashMap<Character,Integer> target=new HashMap<>();//目标元素
        HashMap<Character,Integer> window=new HashMap<>();//滑动窗口
        List<Integer> list=new LinkedList<>();
        char f=p.charAt(0);
        int valid=0;
        for(Character c:p.toCharArray()) target.put(c,target.getOrDefault(c,0)+1);//把p中字符都放入到target中
        while(right<sLen)
        {
            char c=s.charAt(right);
            right++;
            if(target.containsKey(c))//判断这个字符在p中是否出现过
            {
                window.put(c,window.getOrDefault(c,0)+1);//如果出现过，就放到滑动窗口中
                if(target.get(c).equals(window.get(c))) valid++;//如果每一个字符在滑动窗口中出现的次数和target中出现的次数相等，就让valid++，这说明了窗口中放入了valid个有效字符
            }
            if(right-left==pLen)//如果窗口大小等于P的长度，那么判断一下，这个窗口中的字符是不是P的异位词
            {
                char d=s.charAt(left);
                if(valid==target.size()) //如果valid==target.size()，说明了P中所有字母都放入到window中
                {
                    list.add(left);
                }
                //无论该窗口中元素是不是P的异位词，下面都需要开始向右移动滑动窗口
                if(target.containsKey(d))//如果字符d包含在target中
                {
                    if(target.get(d).equals(window.get(d))) valid--;//如果这个字符在target中出现过的次数和window中出现的次数相等，那么向右移动后，window中就少了一个符合要求的字符，valid需要减一
                    window.put(d,window.get(d)-1);//窗口向右移动，所以需要把该字符从窗口中移出
                }
                left++;//向右移动
            }
        }
        return list;//返回结果集
    }
}
